## Projets proposés

- `projet-bootloader/` : séquence de démarrage, mise à jour firmware.
- `projet-commande-moteur/` : boucle de contrôle (PWM, feedback, sécurité).
- `projet-iot-embarque/` : capteurs + stack réseau (ex. LwIP, MQTT).
- `projet-custom/` : template pour ton propre projet.

## Objectif

Être capable de concevoir, documenter, tester et maintenir un système embarqué de bout en bout.
