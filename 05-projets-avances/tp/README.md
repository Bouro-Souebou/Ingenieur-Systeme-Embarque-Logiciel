# TP – Projets avancés

Ce module repose sur un **mini-projet** plutôt que sur des exercices isolés.

## Projet type

Utiliser le template et les consignes du dossier **projet-custom** :

- [../projet-custom/README.md](../projet-custom/README.md)

## Checklist mini-projet

1. Copier `projet-custom` et le renommer (ex. `mon-datalogger`).
2. Définir objectif et fonctionnalités (une phrase + liste courte).
3. Choisir la cible (carte + MCU), créer le projet (CubeMX ou Makefile/CMake).
4. Implémenter par couches : BSP → drivers → app.
5. Documenter build, flash et usage (commandes UART, etc.).
6. Références : cours 5.1 (architecture), 5.2 (tests/débogage), 5.3 (robustesse/sécurité).
