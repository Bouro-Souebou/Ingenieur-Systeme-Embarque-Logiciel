# Glossaire – Systèmes embarqués

| Terme | Signification |
|-------|----------------|
| **ADC** | Analog-to-Digital Converter : convertit une tension en valeur numérique. |
| **BSP** | Board Support Package : couche logicielle propre à une carte (broches, init). |
| **CMSIS** | Cortex Microcontroller Software Interface Standard : API et headers ARM pour Cortex-M. |
| **DMA** | Direct Memory Access : transfert de données entre périphérique et RAM sans CPU. |
| **GPIO** | General Purpose Input/Output : broche configurable en entrée ou sortie. |
| **HAL** | Hardware Abstraction Layer : couche d’abstraction du fabricant (ex. STM32 HAL). |
| **ISR** | Interrupt Service Routine : fonction appelée lorsqu’une interruption se déclenche. |
| **NVIC** | Nested Vectored Interrupt Controller : contrôleur d’interruptions du cœur ARM. |
| **PWM** | Pulse Width Modulation : signal carré dont on règle le rapport cyclique. |
| **RTOS** | Real-Time Operating System : système d’exploitation temps réel (ex. FreeRTOS). |
| **UART** | Universal Asynchronous Receiver-Transmitter : communication série asynchrone. |
| **WCET** | Worst-Case Execution Time : pire temps d’exécution (pour l’analyse temps réel). |

*À compléter au fil de la formation.*
