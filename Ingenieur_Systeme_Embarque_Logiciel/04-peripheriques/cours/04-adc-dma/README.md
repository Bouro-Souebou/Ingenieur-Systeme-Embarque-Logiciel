# ADC et DMA – Vue d'ensemble

Ce sous-dossier contient le cours **ADC et DMA** : principes de l'ADC, conversion one-shot et mode continu/scan, utilisation du DMA pour transférer les résultats en RAM, et bonnes pratiques.

## Objectifs pédagogiques

- Comprendre les **principes** de l'ADC (résolution, canaux, trigger, référence).
- Lancer des **conversions** en one-shot (polling ou interruption).
- Utiliser le **mode continu** et **scan** (plusieurs canaux).
- Utiliser le **DMA** pour transférer les résultats ADC vers la RAM sans charge CPU.
- Appliquer les bonnes pratiques (impédance, bruit, temps d'échantillonnage).
