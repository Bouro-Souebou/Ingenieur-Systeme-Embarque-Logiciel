# Timers, PWM et capture – Vue d'ensemble

Ce sous-dossier contient le cours **Timers, PWM et capture** : principe du timer, SysTick, génération PWM (fréquence, rapport cyclique), et input capture (mesure de période ou largeur d'impulsion).

## Objectifs pédagogiques

- Comprendre le **principe** d'un timer (compteur, overflow, comparaison).
- Connaître le rôle du **SysTick** (souvent utilisé par FreeRTOS).
- Générer un signal **PWM** (ARR, CCR, broche timer) pour LED, moteur, servo.
- Utiliser la **capture** (input capture) pour mesurer période ou largeur d'impulsion.
